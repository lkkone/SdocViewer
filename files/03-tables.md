## 复杂表格

| 功能 | 支持情况 | 备注 |
|------|----------|------|
| **粗体文本** | ✅ 支持 | 使用 `**文本**` 语法 |
| *斜体文本* | ✅ 支持 | 使用 `*文本*` 语法 |
| `行内代码` | ✅ 支持 | 使用 `` `代码` `` 语法 |
| [链接](https://example.com) | ✅ 支持 | 使用 `[文本](URL)` 语法 |
| ~~删除线~~ | ✅ 支持 | 使用 `~~文本~~` 语法 |
